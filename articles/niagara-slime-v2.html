1. 概述与核心理论

1.1 什么是非牛顿流体？

牛顿流体（如空气、水）的剪切应力 τ 与剪切速率 γ̇ 呈线性关系，粘度 μ 为常数。然而，非牛顿流体的粘度随剪切速率变化， 表现出复杂的流变学行为。粘液通常属于假塑性流体（Pseudoplastic Fluid）， 即剪切变稀：静止时粘稠，受力流动时粘度急剧下降。

🎯 核心洞察：粘弹性的双重性

粘液的视觉真实感来源于粘弹性（Viscoelasticity）的双重特性： 短时间尺度下表现为弹性固体（可拉丝、回弹），长时间尺度下表现为粘性液体（可流动、变形）。 这种"记忆效应"需要通过Maxwell 模型或Oldroyd-B 模型描述， 在 Niagara 中通过松弛时间 λ 控制弹性响应的衰减速度。

3. 非牛顿流体核心理论

3.1 幂律模型（Ostwald-de Waele）

描述非牛顿流体最简单的数学模型是幂律模型，它将剪切应力与剪切速率通过幂函数关联。 该模型虽然简单，但对于大多数剪切变稀流体能提供合理的近似。

幂律本构方程：
τ = K · (γ̇)ⁿ
其中有效粘度定义为：
μ_eff(γ̇) = τ / γ̇ = K · (γ̇)ⁿ⁻¹
物理意义：
• n = 1：牛顿流体，粘度恒定 μ = K
• n < 1：剪切变稀（假塑性），粘度随剪切增加而降低。如：番茄酱 n ≈ 0.5，血液 n ≈ 0.8
• n > 1：剪切增稠（胀塑性），粘度随剪切增加而升高。如：玉米淀粉混合物（Oobleck）
🔬 深度技术分析

幂律模型的物理直觉是：在高剪切速率下，流体内部的微观结构（如聚合物链、悬浮颗粒） 沿流动方向排列，减少了内摩擦，导致表观粘度下降。对于粘液特效， 静止时高粘度保持形状，飞溅时低粘度允许展开，这种剪切依赖的粘度变化 正是视觉效果的核心。典型参数范围：K ∈ [10, 100] Pa·sⁿ，n ∈ [0.3, 0.7]。

3.2 Herschel-Bulkley 模型：屈服应力

幂律模型假设任意小的剪切应力都会产生流动。然而，许多真实流体（包括浓稠粘液） 具有屈服应力（Yield Stress） τ₀：只有当应力超过此阈值时才开始流动。 Herschel-Bulkley 模型将 Bingham 塑性模型与幂律模型结合，是描述粘液最完善的模型之一。

Herschel-Bulkley 本构方程：
当 |τ| ≤ τ₀ 时：
γ̇ = 0 （刚体行为，不流动）
当 |τ| > τ₀ 时：
τ = τ₀ · sign(γ̇) + K · (γ̇)ⁿ
模型特性：
• τ₀ 为屈服应力，需克服此应力才能启动流动
• 当 τ₀ → 0 时，退化为幂律模型
• 当 n = 1 时，退化为 Bingham 塑性模型
• 适用于：牙膏、蛋黄酱、浓稠胶水、岩浆等
💡 算法实现注意

在数值实现中，纯屈服应力的处理容易产生刚性问题。实际工程中常使用正则化的 Herschel-Bulkley 模型： 引入小参数 ε，在零剪切附近用近似公式平滑过渡，避免数值奇异性。 在 Niagara 中，可通过检测粒子速度大小动态切换粘性行为。

3.3 Maxwell 粘弹性模型

幂律和 Herschel-Bulkley 模型仅描述粘性行为。真实的粘液还具有弹性： 快速拉伸时会回缩，像弹簧一样。Maxwell 模型将流体视为弹簧（弹性元件） 与阻尼器（粘性元件）的串联组合，是描述粘弹性的最简模型。

Maxwell 本构方程：
τ + λ · (dτ/dt) = μ · γ̇
其中松弛时间 λ = μ / G，μ 为粘度，G 为剪切模量。
时间尺度分析：
• 短时效（t ≪ λ）：弹性主导，dτ/dt 项重要，流体表现为固体，可储存变形能
• 长时效（t ≫ λ）：粘性主导，τ 达到稳态，流体表现为牛顿流体
• 松弛时间 λ 决定了"记忆"的持续时间，粘液典型值 λ ∈ [0.1, 1.0] s

对于更复杂的粘弹性行为（如应变硬化），需要使用Oldroyd-B 模型或 Giesekus 模型，引入额外的结构张量描述聚合物链的拉伸与取向。 但在实时渲染中，Maxwell 模型已能提供足够的视觉效果。

4. SPH 光滑粒子流体动力学

4.1 核函数插值原理

SPH 是一种无网格拉格朗日方法，将流体离散为携带质量、动量等物理量的粒子。 任意场量在空间某点的值通过核函数加权邻域粒子贡献来近似。

SPH 核心公式：
场量插值：
A(x) = Σⱼ mⱼ · (Aⱼ/ρⱼ) · W(|x - xⱼ|, h)
密度计算（核心）：
ρᵢ = Σⱼ mⱼ · W(|xᵢ - xⱼ|, h)
压力梯度（Navier-Stokes）：
∇pᵢ = Σⱼ mⱼ · ((pᵢ + pⱼ)/(2ρⱼ)) · ∇W(|xᵢ - xⱼ|, h)
粘性 Laplacian：
∇²vᵢ = Σⱼ (mⱼ/ρⱼ) · (vⱼ - vᵢ) · ∇²W(|xᵢ - xⱼ|, h)

4.2 常用核函数

Poly6 核函数（用于密度）：
W(r,h) = (315/(64πh⁹)) · (h² - r²)³ , r ≤ h
特性：在 r=0 处非零，适合密度插值；一阶导在 r=0 处为零，避免粒子聚集。
Spiky 核函数（用于压力梯度）：
∇W(r,h) = -(45/(πh⁶)) · (h - r)² · (r/|r|) , r ≤ h
特性：梯度在 r→0 时非零，避免粒子配对不稳定（Pairing Instability）。

4.3 非牛顿 SPH 算法

在标准 SPH 框架中引入非牛顿本构，关键是在每个时间步根据局部剪切速率计算有效粘度。

非牛顿 SPH 单步算法：
对于每个粒子 i：
    
    // 1. 邻居搜索（空间哈希/均匀网格加速）
    Neighbors ← SpatialQuery(xᵢ, h)
    
    // 2. 计算密度
    ρᵢ ← Σⱼ mⱼ · W(rᵢⱼ, h)
    
    // 3. 计算压力（弱可压缩状态方程）
    pᵢ ← k · ((ρᵢ/ρ₀)^γ - 1)
    
    // 4. 计算速度梯度（剪切速率）
    ∇vᵢ ← Σⱼ (mⱼ/ρⱼ) · vⱼᵢ ⊗ ∇W(rᵢⱼ, h)
    γ̇ᵢ ← √(2 · Σ (∇vᵢ)²)   // 第二不变量
    
    // 5. 有效粘度（幂律模型）
    μ_eff ← K · (γ̇ᵢ + ε)^(n-1)   // ε 防止除零
    
    // 6. 计算力
    F_pressure ← -Σⱼ mⱼ · (pᵢ + pⱼ)/(2ρⱼ) · ∇W(rᵢⱼ, h)
    F_viscous ← Σⱼ mⱼ · (μ_eff/ρⱼ) · (vⱼ - vᵢ) · ∇²W(rᵢⱼ, h)
    F_gravity ← ρᵢ · g
    F_surface ← α · Σⱼ mⱼ · (xᵢ - xⱼ) · W_cohesion(rᵢⱼ, h)
    
    // 7. 时间积分（Velocity Verlet 或 PBD）
    vᵢ(t+Δt) ← vᵢ(t) + Δt · (F_total/ρᵢ)
    xᵢ(t+Δt) ← xᵢ(t) + Δt · vᵢ(t+Δt)

CFL 条件：Δt ≤ CFL · h / v_max  （通常 CFL = 0.4）

5.2 关键模块实现

Niagara Custom HLSL - 非牛顿粘度计算：
// Niagara Simulation Stage - Custom HLSL
// Input Attributes
float3 Position;           // 当前粒子位置
float3 Velocity;           // 当前粒子速度  
float Mass;                // 粒子质量
float SmoothingRadius;     // 光滑核半径 h
float ConsistencyK;        // 稠度系数 K
float PowerIndexN;         // 幂律指数 n
float YieldStress;         // 屈服应力 τ₀

// Output
float3 ViscosityForce;     // 粘性力
float EffectiveViscosity;  // 有效粘度（用于调试可视化）

// 辅助函数：Spiky 核梯度
float3 SpikyGradient(float3 r, float h)
{
    float len = length(r);
    if (len > h || len < 0.001) return 0;
    float coeff = -45.0 / (PI * pow(h, 6));
    return coeff * pow(h - len, 2) * (r / len);
}

// 辅助函数：Viscosity Laplacian
float ViscosityLaplacian(float r, float h)
{
    if (r > h) return 0;
    return 45.0 / (PI * pow(h, 6)) * (h - r);
}

// 主函数：计算剪切速率和有效粘度
void CalculateNonNewtonianViscosity()
{
    float3 velocityGradient = 0;
    float shearRateSq = 0;
    
    // 遍历邻居粒子计算速度梯度
    for (int j = 0; j < NeighborCount; j++)
    {
        int neighborIdx = NeighborIndices[j];
        float3 r = Position - Positions[neighborIdx];
        float3 vDiff = Velocity - Velocities[neighborIdx];
        
        float3 gradW = SpikyGradient(r, SmoothingRadius);
        velocityGradient += (Masses[neighborIdx] / Densities[neighborIdx]) 
                           * vDiff * gradW;
    }
    
    // 计算剪切速率（第二不变量的平方根）
    shearRateSq = dot(velocityGradient, velocityGradient);
    float shearRate = sqrt(shearRateSq) + 0.001; // 防止除零
    
    // 幂律模型：μ_eff = K · γ̇^(n-1)
    EffectiveViscosity = ConsistencyK * pow(shearRate, PowerIndexN - 1.0);
    
    // Herschel-Bulkley 屈服应力处理
    float shearStress = EffectiveViscosity * shearRate;
    if (shearStress < YieldStress)
    {
        // 低于屈服应力，近似刚体，极大粘度
        EffectiveViscosity = 10000.0;
    }
    
    // 计算粘性力
    ViscosityForce = 0;
    for (int j = 0; j < NeighborCount; j++)
    {
        int neighborIdx = NeighborIndices[j];
        float3 r = Position - Positions[neighborIdx];
        float dist = length(r);
        
        float3 vDiff = Velocities[neighborIdx] - Velocity;
        float laplacianW = ViscosityLaplacian(dist, SmoothingRadius);
        
        ViscosityForce += Masses[neighborIdx] * (EffectiveViscosity / Densities[neighborIdx])
                        * vDiff * laplacianW / Densities[neighborIdx];
    }
    
    // 应用力到加速度
    Acceleration += ViscosityForce / Mass;
}

5.3 PBD (Position Based Dynamics) 积分

传统 SPH 使用显式积分，受 CFL 条件限制时间步长。Niagara 采用XPBD (Extended Position Based Dynamics)，通过约束投影实现隐式积分，大幅提高稳定性。

XPBD 算法流程：
// XPBD 单时间步
void XPBD_Step(float dt)
{
    // 1. 预测位置（显式欧拉）
    for each particle i:
        v_i_pred = v_i + dt * (F_ext / m_i)
        x_i_pred = x_i + dt * v_i_pred
    
    // 2. 约束投影（迭代求解）
    for iteration = 0 to SolverIterations:
        // 密度约束（不可压缩性）
        for each particle i:
            C = (rho_i / rho_0) - 1  // 密度约束函数
            if |C| > epsilon:
                // 计算约束梯度
                grad_C = (1/rho_0) * sum(m_j * grad_W_ij / rho_j)
                // XPBD 修正
                lambda = -C / (dot(grad_C, grad_C) + alpha/dt^2)
                // 应用位置修正
                delta_x = lambda * grad_C
                x_i_pred += delta_x
        
        // 碰撞约束
        for each particle i:
            if 碰撞检测(x_i_pred):
                x_i_pred = 投影到表面(x_i_pred)
    
    // 3. 更新状态
    for each particle i:
        v_i = (x_i_pred - x_i) / dt
        x_i = x_i_pred
        
        // 阻尼
        v_i *= (1.0 - Damping)
}

6. 次表面散射材质渲染

6.1 粘液的光学特性

粘液视觉效果的关键在于次表面散射（Subsurface Scattering, SSS）： 光线进入介质内部，经过多次散射后从其他位置出射。这种效果使粘液呈现半透明的 "果冻"质感，背光时尤为明显。

SSS 渲染方程：
L_o(x, ω_o) = ∫_Ω [f_BRDF · L_i(x, ω_i) · cosθ_i dω_i] + ∫_A [R(x, x') · L_i(x', ω_i) dA]
第一项：表面反射（BRDF）
第二项：次表面散射（BSSRDF），R(x, x') 为扩散反射核

7.2 性能优化策略

Niagara 流体性能调优

  • ▸ 粒子数控制：移动端 1K-2K，桌面端 5K-10K，视距 LOD 减少远距离粒子
  • ▸ 光滑核半径：h = 2×粒子间距，平衡精度与邻居数量
  • ▸ 空间哈希：网格尺寸等于 h，O(1) 邻居查询替代 O(n²)
  • ▸ PBD 迭代：3-5 次约束投影，迭代越多越稳定但越慢
  • ▸ 时间步长：固定 Δt = 0.016s（60fps），子步进处理高速粒子
